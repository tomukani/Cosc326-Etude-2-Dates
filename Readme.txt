To run program, in terminal navigate to file location and enter command
python3 dates.py
The program will then wait for an input date to be entered and will format it

If you wish to input a text file of line separated date inputs enter the command
python3 dates.py <filename>.txt

If you wish to run the programs doc tests enter the command
python3 dates.py -t -v

Also included are the test file used in testing, text.txt and answers.txt.
test.txt contains a range of randomly generated dates in different valid input formats and
answers.txt contains the expected output for each date in test.txt
